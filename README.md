# Ibiza Home Meeting - Demos IA de Gecko Studio

Repositorio de aplicaciones con Inteligencia Artificial para presentar en el Ibiza Home Meeting.

## Objetivo

Mostrar soluciones prácticas de IA aplicadas al sector de construcción, arquitectura y diseño, diseñadas para generar networking, visibilidad y oportunidades de negocio.

## Proyectos Demo

### 1. Visualización Arquitectónica con IA
**Carpeta:** `/visualizacion-arquitectonica`
- Sube fotos o planos y genera renders 3D realistas
- Propuestas de diseño en diferentes estilos al instante
- **Estado:** 🚧 En desarrollo

### 2. Networking Inteligente
**Carpeta:** `/networking-inteligente`
- Match automático de oportunidades entre asistentes
- Análisis de perfiles complementarios (constructores, arquitectos, proveedores)
- Conexiones estratégicas basadas en proyectos
- **Estado:** 🚧 En desarrollo

### 3. Calculadora de Presupuestos IA
**Carpeta:** `/calculadora-presupuestos`
- Estimación de costes y materiales mediante IA
- A partir de descripción o imágenes del proyecto
- Cálculo de tiempos de ejecución
- **Estado:** 🚧 En desarrollo

### 4. Chatbot Normativa Ibiza
**Carpeta:** `/chatbot-normativa`
- Consultas sobre regulaciones de construcción en Ibiza
- PTI, certificaciones sostenibles
- Asesoramiento en normativa específica
- **Estado:** 🚧 En desarrollo

### 5. Generador de Propuestas Comerciales
**Carpeta:** `/generador-propuestas`
- Crea presentaciones personalizadas en minutos
- Propuestas comerciales automatizadas
- Optimizado para cierre rápido en ferias
- **Estado:** 🚧 En desarrollo

## Instalación y Uso

1. Clonar el repositorio:
```bash
git clone https://github.com/seo-geckostudio/ibizahomemeeting.git
cd ibizahomemeeting
```

2. Abrir `index.html` en el navegador para ver el showcase completo

3. Cada proyecto tiene su propia carpeta con su demo independiente
